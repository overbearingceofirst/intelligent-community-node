# intelligent-community-node

Node.js backend for Intelligent Community (express + MySQL + JWT)

## Features (scaffold)

- User registration & login (JWT)
- Basic models: users, houses, repair orders, payments, visitors
- Authentication & role middleware
- SQL init script: database/init.sql

## Quick Start

1. Install dependencies

```bash
npm install express mysql2 dotenv jsonwebtoken bcryptjs body-parser
```
